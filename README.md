# Pygame Space Shooter
### A simple 2D game in which you must defend the mother ship from rouge asteroids and enemy spaceships
## Dependencies and Use
### This project uses pygame. Install with `pip install pygame`. Pygame's pretty cool, check them out:
![Static Badge](https://img.shields.io/badge/Documentation-black?logo=python&link=https%3A%2F%2Fwww.pygame.org%2Fdocs%2F)
![Static Badge](https://img.shields.io/badge/Discord%20Server-black?logo=discord&logoColor=%235865F2&link=https%3A%2F%2Fdiscord.com%2Finvite%2Fpygame)
![Static Badge](https://img.shields.io/badge/Subreddit-black?logo=reddit&logoColor=%23Ff4500&link=https%3A%2F%2Fwww.reddit.com%2Fr%2Fpygame%2F)
### Please run this from the root of the folder. Run with `python code/main.py`.
